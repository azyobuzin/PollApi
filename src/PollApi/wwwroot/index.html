ツイートを表示

使い方

https://azyobuzin-twpollapi.azurewebsites.net/v1/ツイートID
にアクセスすると、 JSON が返って来ます。例えばこんな感じ 
{
    choice_count: 4,
    end_time: "2015-12-14T06:43:09Z",
    tweet_id: "675928931250253824",
    choices: ["🍤", "🍆", "✊", "✋"]
}
簡単ですね。さらに、 ?detailed=true を指定すると、現在の投票状況も取得できます。 
{
    is_open: "true",
    total: 5,
    percentages: [20, 60, 0, 20],
    choice_count: 4,
    end_time: "2015-12-14T06:43:09Z",
    tweet_id: "675928931250253824",
    choices: ["🍤", "🍆", "✊", "✋"]
}
あんまり大量に叩くと Azure や Twitter に怒られかねないので、常識の範囲内でお願いします。特に detailed の結果はキャッシュしていないので。
